/* --------------------------------------html--------------------------------------- */
DTD：
html 4.01：三种（严格的、普通的、框架的）
XHTML1.0：三种

例：<!DOCTYPE html PUBLIC 
"-//W3C//DTD XHTML 1.0 Transitional//EN" 
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

meta：
元配置：项目的基本配置
注：文件编码和解释编码要一致

例：<meta http-equiv="Content-Type" content="text/html;charset=gb2312"/>

例（关键字）：<meta name="Keywords" content="xx,cc,xx"/>
例（页面描述）：<meta name="Description" content="This is a description"/>

标签类型：
html分类：
容器级：
* div、ul、dl、ol、h
* 里面可以放置任何东西
文本级：
* a、img、form、span、p
* 里面只能放置文字、图片、表单元素（这些都属于文字）


css分类：
块级元素（所有的容器级，加上p）：
* div、ul、dl、ol、h、p
* 霸占一行，不能与其他任何元素并列
* 能接受宽、高
* 如果不设置宽度，那么宽度就是父亲的100%
* 转为行内元素：display:inline;
行内元素（所有文本级，除了p）：
* a、img、form、span
* 有空白折叠现象
* 与其他行内元素并排
* 不能设置宽、高
* 转为块级元素：display:block;



图片：
例：<img src="1.jpg" alt=""/>

src：图片地址
alt：替代文本（图片无法显示的时候）

超链接：
例：<a href="https://www.baidu.com" title="悬停文本" target="_blank">百度</a>

target：是否在新窗口打开（_blank在新窗口打开）

注：可作为锚点
<a href="#xxx"></a>
<a name="xxx" id="xxx"></a>
地址可写为：file:///C:/WebApplication1/index.html#xxx

注：超链接伪类
/* 访问之前 */
a:link {
    color:red;
}
/* 访问之后 */
a:visited {
    color:orange;
}
/* 鼠标悬停 */
a:hover {
    color:blue;
}
/* 点击的时候 */
a:active {
    color:green;
}

/* --------------------------------------css--------------------------------------- */
<style type="text/css"></style>
常用的属性：
color			颜色
font-size			字体大小
font-weight		字体加粗
normal	正常
bold		加粗
font-stype		字体
normal	正常
italic		斜体
font-family
"consols","Arial","Times New Roman"
"宋体","微软雅黑"
text-decoration	字体描述
none	无下划线
underline	下划线


background-color	背景色
background-image（:url(xxx.jpg);）
background-repeat（是否重复，no-repeat、repeat-x、repeat-y）
background-position（center top）（水平居中，顶部对其）
background-attachment（fix固定）
综合就是：
font:14px/24px "宋体";
background:red url(xxx.jpg) no-repeat center center fix;


box-shadow: 0 0 8px #898580;（阴影，x偏移、y偏移、阴影大小、颜色）


选择器（类上样式，id上行为）：
* 标签选择器：直接写标签
* id选择器：#id
* 类选择器：.class（多个类用空格隔开，为了复用）
* 交集选择器：div.class（表示两个交集，一般以标签开头，类跟后）
* 并集选择器：div,li（表示两个并集，逗号链接）
* 后代选择器：.class .li p（后代用空格隔开，某个东西下面的公共东西）

css继承性：
color、text-开头、line-开头、font-开头的都可以被继承
css层叠性：
分别比较id选择器、类选择器、标签选择器数量
id选择器 > 类选择器 > 标签选择器
如果权重一样，谁写在后面听谁的
注：提高权重 font-size:60px !important;


盒子模型（盒子的真实宽度是 width + 2 * padding + 2 * border）：
width
height
padding（上、右、下、左）
margin（margin:0 auto;盒子就居中了，本质是上下0，左右auto）
border（1px dashed red（solid：实线；dashed：虚线；））

注：
1.使用margin:0 auto;，必须有width
2.只有标准流的盒子，才可以使用margin:0 auto;
3.margin:0 auto;是盒子居中，文本居中是text-align:center;，再设置行高为
注：标准流中，margin top和bottom有塌陷现象，以比较大的为准

块级元素和行内元素互转：
display:inline;（转为行内元素）
display:block;（转为块级元素）

脱标的三种手段（脱离标准文档流）：
1.浮动
* float:left;
* 父亲不会被浮动的儿子撑出高的
* 有左边贴靠的
* 浮动的周围有字围现象
* 清除浮动的方法
* 父亲增加高度（很多地方不用高度）
* clear:both;（margin会失效）
* 隔墙法：中间加div，clear:both;，用自身高度达到margin效果
* 内墙法：在上面盒子里添加div，clear:both;，用自身高度达到margin
* 父亲加上overflow:hidden;（内容溢出处理）
2.绝对定位
3.固定定位

注：脱标了就不分块级和行内元素，就可以设置宽高

选择器：
基本选择器：
* id选择器
* 类选择器
* 标签选择器
* 通配符选择器（*）
* 交集、并集选择器（逗号、不带）
层级选择器：
* 后代选择器（空格）
* 子代选择器（>，表示所有儿子代）
* 紧邻选择器（+，表示紧挨着的祥和元素）
* 兄弟选择器（~，表示后面的所有兄弟）
过滤选择器（基本过滤）：
$("a:eq(index)")	选中下标为index的元素
$("a:gt(index)")	下标大于index的元素
$("a:lt(index)")		下标小于index的元素
$("a:odd")		序号为奇数
$("a:even")		序号为偶数
$("a:first")		第一个元素
$("a:last")			最后一个元素
过滤选择器（属性过滤）：
$("a[href]")			包含属性href的
$("a[href='itcast']")		属性href为itcast的
$("a[href!='baidu']")		属性href不为baidu的
$("a[href^='web']")		属性href以web开头的
$("a[href$='cn']")		属性href以cn结尾的
$("a[href*='i']")			属性href包含i的
$("a[href][title='我']")	过滤多个属性
筛选选择器：
parent()		找到父标签
children()		孩子标签
first()		第一个元素
last()		最后一个元素
next()		后一元素
prev()		前一个元素
find()		查找元素
eq(index)		选择第index的元素
siblings()		选择兄弟

节点操作：
attr()	设置或返回属性

事件：
.click()				鼠标单击触发事件，参数可选（data，function）
.hover(func1,func2)		鼠标移入调用func1函数，移出调用func2函数
.focus()/.blur()			鼠标聚焦/失去焦点触发事件（不支持冒泡）
.dblclick()				双击触发，同上
.mousedown()/up()		鼠标按下/弹起触发事件
.mousemove()			鼠标移动事件
.mouseover()/out()		鼠标移入/移出触发事件
.mouseenter()/leave()	鼠标进入/离开触发事件*
.focusin()				鼠标聚焦到该元素时触发事件
.focusout()			鼠标失去焦点时触发事件
.change()				表单元素发生改变时触发事件
.select()				文本元素被选中时触发事件
.submit()				表单提交动作触发*
.keydown()/up()		键盘按键按下/弹起触发
.on()					多事件的绑定
.off()					移除事件的绑定
.trigger(“event”)		触发事件event调用
.triggerHandler()		触发事件，不会冒泡，不会触发默认事件
